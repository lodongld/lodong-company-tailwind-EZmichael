Type of service Web

WEB

AI를 도입함으로써 일어나는 다양한 상황들을 제시하고
그에 대해 찬성과 반대로 갈리는 설문조사입니다.

설문자가 AI에 대해 어떤 관점을 가지고 있는지 통계를
가져올 수 있습니다.

다소 딱딱할 수 있는 AI라는 주제를 어린 연령대의
사용자가 좀 더 친근하게 다룰 수 있도록 전체적으로
부드러운 색상을 활용하여 거부감을 줄였습니다.

설문 결과에 따른 유형별 캐릭터를 부여해 서로 공유하
며 같은 유형을 가진 사람들과 결속감을 느낄 수 있는
재미를 제공 하였습니다.

각 문항마다 생각나는 자신의 의견을 기입하여 AI에 대
해 평소에 어떻게 생각하는지 공유할 수 있습니다.

각 문항 상황에 맞는 삽화를 추가하여 설문조사의 이해
를 도왔습니다.

키오스크의 진행 과정을 상세히 알려주어 고객이 의문
점을 가지게 하지 않습니다.

관리자 페이지에서만 보이는 의견 공유 페이지에서 각
문항마다 설문자들이 적은 의견을 볼 수 있습니다.
